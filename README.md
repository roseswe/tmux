# tmux Example Configuration File

## Installation

````
cp tmux.conf  ~/.tmux.conf
tmux
````

## Features

- Nice status line
- Right click context menu
- multiple windows/screens (crtl-B 1..9)
- horizontal and vertical window split
- mouse support
